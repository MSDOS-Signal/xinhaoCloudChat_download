炘灏云聊 (XinhaoChat) —— 功能强大的全栈即时通讯系统。
集成私聊、群聊、朋友圈以及 AI 智能助手，基于 Spring Boot 3 与 Vue 3 构建。

核心功能

💬 即时通讯

  • 实时消息：WebSocket 低延迟传输
  • 多媒体：支持图片、语音、视频、文件
  • 消息撤回：支持 2 分钟内撤回，群主可撤回任意消息
  • Markdown：支持代码高亮与格式化，适合技术交流

👥 好友与群组

  • 好友管理：搜索、添加、请求处理
  • 群组权限：群主踢人、禁言、修改资料
  • 会话管理：置顶重要会话，智能排序

🤖 AI 智能助手

  • 集成 ModelScope (GLM-5) 模型
  • 支持流式对话（打字机效果）
  • 展示推理过程 (Reasoning Content)
  • 代码自动识别与高亮

🌏 朋友圈 (云聊空间)

  • 发布图文动态
  • 点赞、评论互动
  • 实时通知提醒

技术架构

Spring Boot 3.2.3 MySQL 8.0 JPA (Hibernate) Spring Security + JWT WebSocket (STOMP) Vue 3 Vite Electron Element Plus Pinia

下载与安装

⚠️ 重要提示
  • 服务器位于新加坡，连接可能稍慢，请耐心等待。
  • 大图片建议以文件形式发送，以确保传输稳定性。
  • Android 端麦克风权限可能不稳定。
  • 若提示 timeout 超时属正常现象，多尝试几次即可进入。

Linux 版本

适用于 Ubuntu, Debian, CentOS 等

下载 .tar.gz

安装说明：

# 1. 解压安装包
tar -xvf XinhaoChat-0.0.0-linux.tar.gz

# 2. 设置权限
chmod 1777 *

# 3. 运行程序
./frontend --no-sandbox
